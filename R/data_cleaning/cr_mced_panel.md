# Description

This program grabs data from the following URLs and processes them to
generate a panel dataset for analysis of negative spillover in the NHS
Ghallari trial.

[For Oct
2023-Mar2024](https://www.england.nhs.uk/statistics/statistical-work-areas/cancer-waiting-times/monthly-data-and-summaries/2023-24-monthly-cancer-waiting-times-statistics/)

For Apr 2024 - Sep 2024
<https://www.england.nhs.uk/statistics/statistical-work-areas/cancer-waiting-times/monthly-data-and-summaries/2024-25-monthly-cancer-waiting-times-statistics/>
